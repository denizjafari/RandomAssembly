# Stock Time Series Forecasting with TimesFM + DefeatBeta API

This repository demonstrates how to:

- Fetch OHLCV stock data using the **defeatbeta-api**
- Prepare clean time series datasets for analysis and forecasting
- Generate sliding windows for training & inference
- Run **future price forecasts** using Google's **TimesFM (2.5 - 200M, PyTorch)**
- Compare predictions vs. actual values
- Perform basic statistical and exploratory analysis
